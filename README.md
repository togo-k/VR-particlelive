# VR-particlelive

VR空間で楽しむパーティクルライブ体験です。

### 🌟 特徴
- **VRならではの没入感:** 実際には静止していても、視覚効果によって身体が動いているような錯覚（ベクション）を体験できます。
- **幻想的な演出:** パーティクルによる光のパフォーマンスを楽しめます。

### 🎵 使用楽曲
- **あふぁめーしょん / SHUNTA様**
  - [配信元 (DOVA-SYNDROME)](https://dova-s.jp/bgm/play21564.html)

---

## 🕹 遊び方

1. **ゲーム開始:** 画面中央に「START」の文字が表示されます。
2. **操作:** コントローラーから出ているポインターを「START」に合わせます。
3. **決定:** トリガー（人差し指）を押し込むとライブが始まります。
4. **鑑賞:** 基本的には移動や操作は不要です。そのまま視覚体験をお楽しみください。
